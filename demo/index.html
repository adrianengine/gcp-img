Basic Usage

Provide a URL from Google Cloud Storage amd add it on the src attribute.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"></gcp-img>

With Alt text

Remember to provide a descriptive alt text attribute.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          alt="Three people sitting on a lawn and blowing bubbles."></gcp-img>

Defining a size of 200px

To change the width of the image provide a size attribute with the value.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          size="200"></gcp-img>

With Source Set Image Dimensions

For responsive images provide a configuration to the sizes attribute.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          sizes="[
            {'screen':320,'size':320},
            {'screen':600,'size':640},
            {'screen':1024,'size':520}]"></gcp-img>

With Source Set Image Dimensions and a breakpoint art directed

It is possible to pass a source value on the sizes configuration for the size you need.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          sizes="[
            {'screen':320,'size':320},
            {'screen':600,'size':640},
            {'screen':1024,'size':520,'source':'https://lh3.googleusercontent.com/…'}]"></gcp-img>

Change the Cache TTL

In case you need to specify a different cache time to live days, you can use ttl attribute.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          ttl="180"></gcp-img>

Rotate

To rotate the image clockwise use the rotate attribute. Only valid values are 90, 180 and 270


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          rotate="270"></gcp-img>

Flip

It is possible to flip the image using flip attribute. Only valid options are h to flip horizontally and v to flip vertically.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          flip="v"></gcp-img>

Blur

Add some blur to the image with the blur attribute. Pass a value between 0 and 100.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          blur="40"></gcp-img>

Vignette

Add a vignette to the image with the vignette attribute. Pass a value between 0 and 100.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          vignette="100"></gcp-img>

Invert

Invert the image using invert attribute.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          invert="true"></gcp-img>

Black & White

Convert to black and white the image using bw attribute.


          <gcp-img
          src="https://lh3.googleusercontent.com/…"
          bw="true"></gcp-img>
